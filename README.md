# GameHighScore

Contain an implementation to return the high scores of a game

# Start api

```shell
    cd api/
    mvn clean install -Dmaven.test.skip=true
```

# Start ui

```shell
    cd ui/
    npm start
```

# Run docker

```shell
    bash docker.sh
```
